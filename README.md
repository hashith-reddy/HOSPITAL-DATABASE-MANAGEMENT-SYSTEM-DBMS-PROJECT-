Hospital Management System (HMS)

A simple, database-driven Hospital Management System built using Flask,
Python, MySQL, and HTML and XAMPP.

Features

-   User Login & Signup
-   Doctor Management
-   Patient Management
-   Appointment Booking
-   Responsive UI

Tech Stack

-   Backend: Python, Flask, XAMPP
-   Frontend: HTML
-   Database: MySQL

Setup

1.  Create database: CREATE DATABASE hms;

2.  Import hms.sql

3.  Install dependencies: pip install flask mysql-connector-python,pip install flask,pip install Flask-SQLAlchamey,pip install Flask-Mail,pip install mysqlclient, pip install Flask-Login

4.  Run: python main.py

Access

http://127.0.0.1:5000/
